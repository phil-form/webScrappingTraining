Exercices - 02_web_scrapping_base

1) HTTP basics
- Ecrire une fonction qui affiche: schema, host, path, params d une URL.
- Tester avec 2 URLs differentes.

2) Requests GET + timeout
- Faire un GET vers JSONPlaceholder /posts/1.
- Ajouter un timeout et verifier le status.

3) Requests params + headers
- Faire une requete GET vers /posts avec params userId=1.
- Ajouter un User-Agent custom.

4) SQLAlchemy CRUD
- Creer une table posts avec id, title, body.
- Inserer 3 posts puis lire tous les titres.

5) BeautifulSoup
- Parser un HTML local et extraire tous les liens.
- Filtrer les liens qui contiennent example.

6) Scraping responsable
- Ajouter un delai (sleep) entre 2 requetes.
- Ecrire un petit resume des bonnes pratiques.
