Exercices - 01_file_management

1) Parcours et filtres
- Lister tous les fichiers du dossier example/ mais exclure ceux qui finissent par .txt.
- Afficher uniquement les chemins relatifs.

2) Encodage
- Ecrire un fichier avec des caracteres accentues et des emojis.
- Relire le fichier et compter le nombre de caracteres.

3) CSV
- Ecrire un CSV avec colonnes: id, nom, ville.
- Lire le CSV et convertir id en int.

4) JSON
- Ecrire un dictionnaire contenant une liste de 3 personnes.
- Lire le JSON et filtrer les personnes de plus de 30 ans.

5) XML
- Creer un XML avec une balise racine <produits> et 3 <produit>.
- Parser le XML et afficher nom + prix.

6) Generateurs
- Ecrire un generateur qui renvoie les lignes non vides d un fichier.
- Comparer avec une lecture classique (liste).
